# my_vue_blog



## 层级结构：

#### 1.Router层

- 负责将请求路径分配至对应Controller层中对应的处理方法。

#### 2.Controller层

- 类似于Service（业务层），专注业务逻辑

#### 3.Dao层

- 位于较底层的数据访问层，直接与数据库交互，进行增删改查等操作。
